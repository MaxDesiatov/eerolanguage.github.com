eero
objective-c, evolved
Eero is a dialect of Objective-C designed to make developers more productive through improved code readability, maintainability, and safety.
The myth? "It's just syntax." The fact: Readability matters. Code is read many more times than it’s written—and after a few months, even your own code can start to look like someone else’s. Eero’s streamlined syntax and Python-like indentation make code easier to read, understand, and maintain.
time-saving
Poor code readability costs you development time. Syntactic "noise," redundant notation, and boilerplate all distract and provide opportunities for "dumb" and sometimes subtle errors. No language can guarantee clean code, but Eero helps you do a better job.
highly interoperable
Excellent, nearly seamless, interoperability with Objective-C, C, and C++. Just import and use SDK, third-party, and personal frameworks and libraries. And all LLVM memory management choices are available, including manual reference counting and ARC.
high-performance
Compiles down to the same binary code as Objective-C, so you'll enjoy the same performance as standard LLVM-generated Objective-C code. This includes any high-performance C functions or C++ objects you use!
